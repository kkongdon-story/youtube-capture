1. 이 확장은 뭘 하는 건가요?

유튜브 영상을 보고 있을 때, 키보드 한 번 누르면 영상의 모든 말이 글로 정리된 파일로 저장되는 도구입니다.

이런 게 가능해요

  • 📝 자막을 깔끔한 문서(Markdown, PDF, HTML, Excel)로 저장
  • 🌏 영어 영상을 한국어로 자동 번역
  • 🧹 AI가 "어어 그게..." 같은 더듬거림을 자동 정리
  • 💡 영상 내용을 핵심 포인트로 요약
💡 참고: 자막이 있는 영상만 가능합니다 (대부분의 영상에 있어요). AI 기능은 옵션이라 안 써도 됩니다.

2. 설치 (3단계)

2-1. Chrome 확장 설치 (1분)

1
Chrome 주소창에 chrome://extensions 입력 후 엔터
2
우측 상단 "개발자 모드" 토글을 켭니다
3
왼쪽 위 "압축해제된 확장 프로그램을 로드합니다" 클릭
4
내려받은 폴더의 extension 폴더를 선택
5
카드가 나타나면 성공!
✅ 여기까지만 해도 작동해요. 자막 추출은 다른 도구 필요 없음. AI 기능을 안 쓰면 끝!

2-2. 헬퍼 설치 (AI를 쓸 거면 필요, 약 5분)

헬퍼가 뭐냐고요? Chrome 확장이 직접 컴퓨터 프로그램을 실행할 수 없어서, 사이에서 다리 역할을 해주는 작은 프로그램입니다.

1
Node.js 설치 — nodejs.org/ko에서 LTS 다운로드 → 더블클릭 → 다음 → 완료
💡 Node.js가 뭔지 몰라도 괜찮아요. 그냥 깔기만 하면 됩니다.
2
내려받은 폴더에서 운영체제에 맞는 설치 마법사 더블클릭:
  • Windows: installers\windows\install.bat
  • macOS: installers/macos/install.command
    ⚠️ macOS는 처음 열 때 보안 경고가 뜰 수 있어요. 시스템 설정 → 개인정보보호 → "그래도 열기"
3
검은 창이 자동으로 진행됩니다. 가만히 기다리세요.
4
중간에 "Ollama를 자동 설치할까요? (Y/n)" 물으면 Y 입력 후 엔터
✅ "설치 완료" 메시지가 나오면 성공!

3. AI 기능 켜기 (선택, 무료)

자막을 그냥 저장하는 건 위 설치만으로 충분합니다. AI 자동 정리·번역까지 원한다면:

Ollama로 무료 AI 쓰기 (추천)

  • 본인 컴퓨터에서 돌아가서 완전히 무료
  • 인터넷 없어도 됨
  • 속도: 보통 30초~1분

설치 마법사가 자동으로 Ollama와 모델까지 깔아줬다면 끝. 안 깔렸다면 마법사 다시 실행하세요.

또는 ChatGPT Plus / Claude Pro 구독 활용

이미 구독료를 내고 있다면 그것을 쓸 수 있어요. 터미널(Windows: PowerShell)에서:

# ChatGPT Plus 사용
npm install -g @openai/codex
codex login

# 또는 Claude Pro 사용
npm install -g @anthropic-ai/claude-code
claude login
💡 둘 중 하나만 설치하면 됩니다. 빠른 건 Claude Pro의 haiku 모델.

4. 사용법 — 5분 안에 마스터

기본 사용

1
YouTube에서 보고 싶은 영상 열기
2
키보드에서 Ctrl + Shift + S 누르기 (macOS: Cmd)
3
잠시 후 다운로드 폴더의 YouTube-Capture에 파일 저장됨

다양한 포맷으로 저장

확장 아이콘(📊 분석가)을 클릭하면 5가지 포맷 중 선택 가능:

포맷특징언제 쓰면 좋나
Markdown구조 보존Obsidian, 노션, GitHub 등
TXT가장 단순그냥 읽기, 메모장
HTML웹페이지브라우저로 보기 좋게
PDF인쇄 가능저장·공유·인쇄
CSV표 형태Excel로 자막 검색·분석

5. 활용 예시

📚 강의 영상을 노트로 만들기

유튜브 강의를 켜고 Ctrl+Shift+S → Markdown 저장 → Obsidian/노션에 붙여넣기 → 검색 가능한 노트 완성

🌏 영어 영상으로 공부하기

영어 영상을 캡처하면 원본 영어 대본 + 한국어 번역이 자동으로 같이 저장됩니다. 한 줄씩 비교하며 공부하기 좋아요.

📊 인터뷰·세미나 분석

CSV 포맷으로 저장하면 Excel에서 "타임스탬프, 발언, 번역" 3열로 나옵니다. 특정 키워드 검색하거나 통계 내기 좋음.

6. 안 될 때 — 자주 묻는 질문

Q. 단축키를 눌러도 아무것도 안 일어나요

  • YouTube의 영상 페이지에서 눌러야 합니다 (홈, 검색 결과 X)
  • 주소가 youtube.com/watch?v=... 형태인지 확인
  • 다른 확장과 단축키 충돌일 수 있음 → chrome://extensions/shortcuts에서 변경

Q. "자막을 찾을 수 없습니다" 토스트가 떠요

  • 그 영상에 자막이 없는 거예요 (라이브 방송, 음악 영상 일부 등)
  • 영상 설명 더보기에서 "스크립트 표시" 버튼이 있는지 확인. 없으면 자막 자체가 없음

Q. AI가 작동하지 않아요

  • 확장 옵션 페이지 열기 → "지금 점검" 버튼 클릭
  • 각 단계에 ✓ 표시되는지 확인. 빨간 X면 그 단계 다시 진행
  • Ollama가 설치된 경우, PowerShell 새로 열어서 ollama list로 모델 확인

Q. 한글이 깨져 보여요

  • CSV로 저장 시 Excel에서 한글이 깨지면 같은 영상을 다시 캡처해보세요. UTF-8 BOM이 자동 포함됩니다.
  • 다른 포맷(Markdown, TXT)은 거의 안 깨집니다.

Q. 더 자세한 기술 정보가 필요해요

설정 페이지에서 "기술 정보 (고급)" 링크를 클릭하면 헬퍼 진단, 로그 위치 등 자세한 정보가 있습니다.

문제가 해결되지 않으면 GitHub Issues로 신고해주세요. 로그 파일: ~/.youtube-capture/helper.log
